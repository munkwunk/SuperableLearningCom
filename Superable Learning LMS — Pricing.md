## Accessibility Review Add‑On
Affordable accessibility auditing designed for creators and small organizations.  
Typical WCAG audits cost **$75–$120/hr** — we price ours accessibly because accessibility should be accessible.

- **Basic Review** — A quick, high‑level accessibility check that identifies surface‑level issues without WCAG mapping or remediation guidance.  
- **Full WCAG Audit** — A complete diagnostic that maps issues to WCAG 2.2 AA criteria with severity ratings and impact explanations.  
- **Full WCAG Audit + Remediation Plan** — A full WCAG audit plus detailed remediation instructions, code‑level guidance, and regression testing steps.  

### Per Module
When not combined with the full course rate, below.
- **$15** — Basic Review
- **$30** — Full WCAG Audit
- **$40** — Full WCAG Audit + Remediation Plan

### Per Course
A course is equal to five modules (five pages). For longer courses, add the per-page rate.
- **$50** — Basic Review. Add $10 per additional module.
- **$100** — Full WCAG Audit. Add $20 per additional module.
- **$150** — Full WCAG Audit + Remediation Plan. Add $30 per additional module.

### Subscription
Ongoing audit support at our lowest rate. Two month minimum. Cancel any time after that.
For longer courses, add the per-page rate.
- **$45 per month** — Basic Review. Add $10 per additional module.
- **$90 per month** — Full WCAG Audit. Add $20 per additional module.
- **$125 per month** — Full WCAG Audit + Remediation Plan. Add $30 per additional module.

### Accessibility Review Definitions & Scopes

#### Basic Review
A fast, high‑level accessibility check intended for early drafts and lightweight validation.  
**Includes:**  
- Surface‑level accessibility issues  
- Color contrast problems  
- Reading‑order concerns  
- Missing alt text  
- Obvious structural issues  
- General accessibility notes  

**Does NOT include:**  
- WCAG success criteria mapping  
- Severity ratings  
- Remediation instructions  
- Code‑level fixes  
- Component‑level deep analysis  

#### Full WCAG Audit
A complete accessibility diagnostic aligned with WCAG 2.2 AA.  
**Includes:**  
- WCAG 2.2 AA success criteria mapping  
- Severity ratings  
- Impact explanations  
- Component‑level analysis  
- Screenshots or code references  

**Does NOT include:**  
- Remediation instructions  
- Code‑level fixes  

#### Full WCAG Audit + Remediation Plan
A full diagnostic plus a detailed, actionable remediation plan.  
**Includes:**  
- Everything in the Full WCAG Audit  
- Specific remediation steps  
- Code snippets or pseudo‑code  
- Component‑level fixes  
- NVDA/JAWS testing instructions  

#### Subscription Review
A monthly accessibility maintenance plan that offers one course review at our lowest available rate.
Requires a minimum two-month commitment, then cancel any time after that.
