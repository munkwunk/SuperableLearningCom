Welcome to the Accessibility Tree

Before you can fix accessibility issues, you need to understand how assistive technologies see the web. This course will take you behind the curtain to explore the Accessibility Tree.

What You'll Learn

  • Module 1: How the DOM becomes the Accessibility Tree.
  • Module 2: The role of Semantics (Name, Rank, and Role).
  • Module 3: How to inspect the tree using browser DevTools.
  • Module 4: Putting it all together with a final quiz.
