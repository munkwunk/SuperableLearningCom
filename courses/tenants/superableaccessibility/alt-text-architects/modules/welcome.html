Alt-Text Architect

Welcome. You are going to practice writing alternative text (alt text) for images. Using sea cucumbers as our example, you will learn how to choose the right description based on what is happening on the page.

In this activity, you will see several examples from an online course. Your job is to pick the best alt text for the sea cucumber images.

What You Will Learn

  • When to describe an image and when to leave it silent.
  • How to avoid repeating information that is already in the text.
  • How to handle complex charts and diagrams.
  • How to describe what a button or link does.
  • How to describe images that are meant to set a mood.
